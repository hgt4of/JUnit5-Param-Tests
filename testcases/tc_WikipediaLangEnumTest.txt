ТЕСТ-КЕЙС: Переключение языков на Википедии
Автотест: WikipediaLangEnumTest

ПРЕДУСЛОВИЕ:
1. Браузер открыт
2. Переход на https://www.wikipedia.org

ШАГИ:
1. Для каждого языка из набора {English, Français, Deutsch, Italiano, Español}:
   1.1. Нажать на кнопку с названием языка
   1.2. Дождаться загрузки страницы
   1.3. Проверить заголовок вкладки браузера (title)

ОЖИДАЕМЫЙ РЕЗУЛЬТАТ:
- Для English → заголовок содержит "Wikipedia"
- Для Français → заголовок содержит "Wikipédia"
- Для Deutsch → заголовок содержит "Wikipedia"
- Для Italiano → заголовок содержит "Wikipedia"
- Для Español → заголовок содержит "Wikipedia"
